## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/arthurbrenno/tartaruga-vs-lebre.git
```

Entre no diretório do projeto

```bash
  cd caminho_para_o_projeto
```

No diretório do arquivo, abra:

```bash
  clickToRun.bat
```
